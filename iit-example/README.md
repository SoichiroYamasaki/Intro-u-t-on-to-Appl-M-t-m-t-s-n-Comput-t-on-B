# IIT Example: Independent vs Integrated System

This folder contains a Python script that shows a simple example related to Integrated Information Theory (IIT).

## Description

Integrated Information Theory explains consciousness in terms of how much information is integrated in a system.

In this example, I compare two simple systems:

1. Independent system
   - Node A and Node B change independently.

2. Integrated system
   - Node A and Node B influence each other.

The Python script creates a plot comparing these two systems.

## Files

- `plot_iit_example.py`: Python script that generates the plot.
- `README.md`: Explanation of this example.
